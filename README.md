# NFA to DFA

Converts a Non-Deterministic Finite Automata to a Deterministic Finite Automata. Here is an example from the file `input.json`

```
+---------------------+--------------------+--------+--------------+
|                     |         a          |   b    |      c       |
+---------------------+--------------------+--------+--------------+
|      ->('s0',)      |    ['s1', 's2']    |   []   |      []      |
|    ('s1', 's2')     | ['s2', 's3', 's4'] | ['s1'] | ['s1', 's2'] |
| *('s2', 's3', 's4') | ['s2', 's4', 's5'] | ['s1'] |    ['s2']    |
| *('s2', 's4', 's5') | ['s2', 's4', 's5'] | ['s1'] |    ['s2']    |
|       ('s1',)       |       ['s3']       | ['s1'] |    ['s1']    |
|       ('s3',)       |       ['s5']       |   []   |      []      |
|      *('s5',)       |         []         |   []   |      []      |
|       ('s2',)       |    ['s2', 's4']    | ['s1'] |    ['s2']    |
|    *('s2', 's4')    | ['s2', 's4', 's5'] | ['s1'] |    ['s2']    |
+---------------------+--------------------+--------+--------------+
+-----+---+---+---+
|     | a | b | c |
+-----+---+---+---+
| ->A | B | - | - |
|  B  | C | E | B |
| *C  | D | E | H |
| *D  | D | E | H |
|  E  | F | E | E |
|  F  | G | - | - |
| *G  | - | - | - |
|  H  | I | E | H |
| *I  | D | E | H |
+-----+---+---+---+
```
